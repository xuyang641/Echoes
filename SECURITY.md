# Security Policy

## Supported Versions

We currently support the latest version of Echoes.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Echoes seriously. If you discover a security vulnerability, please do NOT open an issue. Instead, please contact us directly.

### How to Report

Please email us at **2311752562@qq.com** with a description of the vulnerability and how to reproduce it. We will acknowledge receipt of your report within 48 hours and provide an estimated timeframe for a fix.

### Policy

*   We ask that you do not publicly disclose the issue until we have had a chance to fix it.
*   We will do our best to respond to your report quickly and work with you to resolve the issue.
*   We appreciate your help in making Echoes more secure!
