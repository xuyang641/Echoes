# 爱发电 (Afdian) 方案配置指南

本文档旨在指导您如何在爱发电后台配置与 Echoes App 逻辑相匹配的赞助方案。

## 核心逻辑说明

Echoes 的后端 Webhook (`afdian-webhook`) 是根据**支付金额**来自动判定会员类型的。因此，您在爱发电后台创建的方案金额必须严格对应：

*   **月度会员 (Monthly)**: 支付金额 **≥ ¥10.00** 且 **< ¥96.00**
*   **年度会员 (Yearly)**: 支付金额 **≥ ¥96.00**
*   **纯赞助 (Donation)**: 支付金额 **< ¥10.00** (仅记录，不发放会员)

---

## 推荐方案配置

请登录 [爱发电创作者后台](https://afdian.com/dashboard)，进入 **“设置” -> “赞助方案”**，创建以下三个方案：

### 方案一：Pro 月度会员 (基础档)

这是用户订阅月费会员的入口。

*   **方案名称**: `Pro 月度会员` (或 `Echoes Pro Monthly`)
*   **金额**: `¥10.00`
*   **方案封面**: (建议上传一张带有 Echoes Logo 的图片)
*   **方案介绍**:
    ```text
    解锁 Echoes Pro 所有高级功能 (1个月)：
    
    ✅ 无限云端同步与备份
    ✅ 高级 AI 智能分析与绘图 (每日100次)
    ✅ 解锁所有 3D 沉浸式主题
    ✅ 4K 无损导出与矢量 PDF
    ✅ 优先技术支持
    
    ⚠️ 重要：支付时请务必在“留言”处填写您的 Echoes 用户 ID，否则无法自动到账！
    ```
*   **独立库存**: 不限
*   **开启售卖**: 是

### 方案二：Pro 年度会员 (优惠档)

这是用户订阅年费会员的入口，相比月付节省 20%。

*   **方案名称**: `Pro 年度会员` (或 `Echoes Pro Yearly`)
*   **金额**: `¥96.00`
*   **方案封面**: (建议上传一张金色的 Echoes Logo 图片)
*   **方案介绍**:
    ```text
    解锁 Echoes Pro 所有高级功能 (12个月) - 立省 20%！
    
    ✅ 无限云端同步与备份
    ✅ 高级 AI 智能分析与绘图 (每日100次)
    ✅ 解锁所有 3D 沉浸式主题
    ✅ 4K 无损导出与矢量 PDF
    ✅ 优先技术支持
    ✅ 专属“创始人”徽章
    
    ⚠️ 重要：支付时请务必在“留言”处填写您的 Echoes 用户 ID，否则无法自动到账！
    ```
*   **独立库存**: 不限
*   **开启售卖**: 是

### 方案三：请我喝咖啡 (纯赞助)

用于用户自愿打赏，金额不限（但建议设置一个低门槛）。

*   **方案名称**: `请开发者喝咖啡`
*   **金额**: `¥1.00` (用户可以自定义增加)
*   **方案介绍**:
    ```text
    纯粹的支持与鼓励！您的支持将用于服务器维护和新功能开发。
    
    ❤️ 感谢您的慷慨！
    (注：此方案仅为赞助，不包含 Pro 会员权益。如需会员请选择上方方案)
    ```

## 关键配置检查

1.  **Webhook 设置**:
    *   确保您已按照 `GUIDE_WEBHOOK.md` 中的步骤，将 Supabase Edge Function URL 填入爱发电的 **“开发者 - Webhook”** 设置中。
    *   勾选 **“订单支付成功”** 事件。

2.  **用户 ID 引导**:
    *   在爱发电的 **“自动回复”** 或 **“置顶动态”** 中，再次提醒用户：**支付时必须在留言区填写 Echoes App 中的 User ID**。
    *   如果用户忘记填写，您需要在爱发电后台手动查看订单，并在 Supabase 数据库中手动为该用户开通权限（或者让用户联系您补填）。

3.  **测试**:
91→    *   配置完成后，建议您自己（或找朋友）用小号测试一笔 10元的订单，确保 Webhook 能正确触发并自动开通会员。

93→## 常见问题
94→
95→*   **Q: 如果用户支付了 ¥10.00 怎么办？**
    *   A: 根据逻辑，¥10.00 ≥ ¥10.00 且 < ¥96.00，系统会将其识别为 **月度会员**，有效期增加 1 个月。
*   **Q: 如果用户支付了 ¥100.00 怎么办？**
    *   A: 根据逻辑，¥100.00 ≥ ¥96.00，系统会将其识别为 **年度会员**，有效期增加 1 年。
*   **Q: 用户忘记填 User ID 怎么办？**
    *   A: Webhook 会记录这笔交易到 `payment_transactions` 表，但在 `user_id` 字段会是空的（或无法匹配）。您需要人工介入处理。
